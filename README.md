-------------->hostapd-python-script<-----------------

by Nimesh Ghelani (nims11)
Python Script to make using and configuring hostapd easier.

==== Dependencies ====
* dhcpd
* hostapd
* python (2.7)

==== Usage ====

chmod +x hostapd.py
sudo ./hostapd.py

This will start hostapd.py in interactive mode.
